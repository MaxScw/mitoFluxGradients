# Heterogeneous flux capacity and oxygen sensitivity lead to subcellular ETC flux gradients in mouse oocytes

Code for reproducing the figures for the paper (**TBD: add preprint link**). All plotting methods are contained within **"plot_generation"**, **"postprocessing"** contains every script necessary to reproduce the underlysing data analysis & modelling. 

For both to work, the **"mitoFluxGradients_data"** repository needs to be downloaded (**TBD: add link**) and located inside the cloned GitHub repository.

In order to run the scripts, enter the "**plot_generation**" directory and set the variable "**project_path**" inside the "**setup.m**" script to the location of the GitHub repository on your machine.

All code was developped & tested under MatLab version 24.2.0.2923080 (R2024b) Update 6
